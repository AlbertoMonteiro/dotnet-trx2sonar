# TRX To Sonar(qube)

Basic dotnet core console app to convert TRX files to Sonarqube Generic Test Data (Generic Execution).

### Usage

* `--help` `-h` `-?`
   Show the current help.
* `-d` [string]
  Solution directory to parse (recursive).
* `-o` [string]  
  Output filename.
* `-a` `--absolute`  
  Use absolute path for Sonarqube file path.
